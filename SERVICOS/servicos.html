Geração de relatórios

DESCRIÇÃO DO SERVIÇO

O serviço de geração de relatórios é o processo de tratamento de uma grande quantidade de dados contido no sistema principal da SOSAnimals, este serviço conta com diversos profissionais especializados para fazer este serviço, contando com diversos tipos de técnicas e programas criados especialmente para este tipo de serviço. Os relatórios são feitos com a principal finalidade que é a de facilitar a analise para uma tomada de decisão.


QUEM PODE SOLICITAR O SERVIÇO

Chefes de departamentos


COMO SOLICITAR

Entre no sistema da SOSAnimal com o cadastro de chefe de departamento, vá ate a equipe SERVIÇOS e selecione o serviço "geração de Relatórios" nela você poderá enviar um pedido de serviço, especificando qual o tipo de relatório, o que precisa conter, e os motivos, este pedido será encaminhado para um profissional que analisara seu pedido e logo entrara em contato.

Instalação de software

DESCRIÇÃO DO SERVIÇO

O serviço de instalação de software tem como principal finalidade é fazer a instalação de softwares requisitados nas maquinas, fazendo também a verificação de segurança deste.


QUEM PODE SOLICITAR O SERVIÇ

Tanto funcionário normais quanto chefes de departamentos


COMO SOLICITAR

O serviço pode ser solicitado no sistema da SOSAnimal, dentro da equipe SERVIÇOS localize o serviço "Instalação de softwares" e envie um pedido, informando o nome do software, motivo desta instalação, versão desejada (caso não seja informado, será instalado a mais recente), este pedido será encaminhado para uma analise e caso seja aceito logo entraram em contado.
